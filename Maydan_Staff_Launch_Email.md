# Maydan Staff Launch Email

## Subject

Maydan is now live for BHA Prep event submissions

## Email Draft

Assalamu alaikum,

Maydan is now live as BHA Prep's internal workspace for staff event submissions, approvals, notifications, and calendar coordination.

Starting now, staff should use Maydan for event requests instead of managing approvals informally over email or chat.

### What Maydan handles

- Event submission by staff
- Automatic approval routing based on entity
- Facilities visibility on all event requests
- Internal approved-events calendar
- Status tracking for submitted events

### What staff should do

1. Log in at `https://maydan-six.vercel.app`
2. Use your BHA Prep email and your Maydan password
3. Open `Events`
4. Submit new events through the event form
5. Check `Approvals` if your role includes approval responsibility

### Notes

- Facilities Director is copied on all event submissions
- Rejected events can be revised and resubmitted
- Only approved events appear on the internal calendar
- Parent/student access and public calendar access are not part of this version

### Support

If something looks incorrect, or if an event seems routed to the wrong approver, report it immediately to the Maydan admin before retrying multiple times.

Jazakum Allahu khayran.
